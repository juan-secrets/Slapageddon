# Slapageddon -- Sprint 4

This repository contains the project developed during the fourth sprint
at **Epitech Berlin**, where we were tasked with creating a simple game
with lots of juice and polish.

## 🎮 Gameplay & Mechanics

Our game will be a game version of the card game "Bataille Corse" (Corsican battle). 

## 📦 Required Asset Packs

The project relies on several free asset packs. If you want to open and
run the project in **UE5** (not just play the build), make sure to
download the following:

-   [52 Card Decks](https://www.fab.com/listings/57e8bc76-6f19-4b5e-bf09-e912eba4c88f)
-   [Wild West Town - Relive the Western Adventures](https://www.fab.com/listings/25d8d2df-3c68-4642-adc9-53169e12ad12)
-   [Old West - VOL 5 - Town Props](https://www.fab.com/listings/216c3025-105f-44e0-ae16-756be31c452d)
-   [Old West - VOL 4 - Goods and Supplies](https://www.fab.com/listings/7ec26cda-15cf-4b62-ac9b-45f156676e51)

## 🚀 Getting Started

-   Clone the repository.
-   Download and install the required asset packs.
-   Open the project in **Unreal Engine 5**.
-   Play the build or explore the project to see how the mechanics are
    implemented.
